Para sincronización en tiempo real entre dispositivos, ingresa tus credenciales de Firebase Realtime Database.

¿Cómo obtener estas credenciales?
1. Ve a console.firebase.google.com
2. Crea un proyecto → Realtime Database → Crear base de datos (modo prueba)
3. En Configuración del proyecto → Aplicaciones web → Registrar app
4. Copia databaseURL, apiKey y projectId

Reglas requeridas en Firebase:
{ "rules": { ".read": true, ".write": true } }
